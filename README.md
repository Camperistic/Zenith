# WoW-TBC-Addons

World of Warcraft: The Burning Crusade Classic (2.5.x) addons.

## Zenith

**Guided ascent to 70 — optimized every step.**

A leveling + min-max companion: a RestedXP-style 1–70 route plus real-time **talent**,
**gear**, **pet**, and **rotation** coaching. Flagship class implemented end-to-end:
**Beast Mastery Hunter**; the engine and UI are data-driven so other classes plug in
as data files.

See **[Zenith/README.md](Zenith/README.md)** for features, install, commands, and the roadmap.
