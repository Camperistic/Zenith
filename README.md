# WoW-TBC-Addons

World of Warcraft: The Burning Crusade Classic (2.5.x) addons.

## Zenith

**Guided ascent to 70 — optimized every step.**

A leveling + min-max companion: a RestedXP-style 1–70 route (all races, both factions)
plus real-time **talent**, **gear**, **pet**, and **rotation** coaching. **All 9 classes**
have a researched leveling talent path and rotation helper; Beast Mastery Hunter is the
deepest slice (gear milestones, pre-raid BiS, pets). Data-driven, MIT-licensed, and set up
for CurseForge / Wago / WoWInterface packaging.

See **[Zenith/README.md](Zenith/README.md)** for features, install, commands, and the roadmap.
