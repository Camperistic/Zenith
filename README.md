# WoW TBC Addons

A collection of addons for **World of Warcraft: Burning Crusade Classic (2.5.x / Anniversary realms)**.

## Addons

| Addon | Description |
|---|---|
| **[TalentMate](TalentMate/)** | Class-aware talent build library + auto-applier. Pick a PvE / PvP / leveling spec and it spends your points for you, including an optimized level-by-level leveling order. |

## Installing

Copy the addon's folder (e.g. `TalentMate/`) into your WoW AddOns directory:

```
World of Warcraft\_anniversary_\Interface\AddOns\
```

Then `/reload` or restart the game. See each addon's own README for usage.
